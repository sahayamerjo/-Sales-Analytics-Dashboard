# 📊 Sales Performance Dashboard in Microsoft Excel

## 📌 Overview

This project showcases an interactive Sales Dashboard built using Microsoft Excel. It is designed to analyze sales data, track key performance indicators (KPIs), and provide meaningful insights to support data-driven decision-making.

The dashboard presents complex data in a simple and visually appealing format using charts, slicers, and Excel functions.

---

## 🎯 Objectives

* Analyze overall sales performance
* Track key business metrics and KPIs
* Identify sales trends and patterns
* Compare performance across regions, products, or time periods
* Provide actionable insights for business growth
